# Sistema Domótico IoT con Home Assistant
Sistema domótico completo con integración de dispositivos IoT, automatización y control inteligente del hogar.

## Descripción

Proyecto de implementación de un sistema domótico completo basado en Home Assistant, con integración de dispositivos IoT y comunicación mediante MQTT.

## Tecnologías utilizadas

* Ubuntu Server
* Docker
* Home Assistant
* Broker MQTT (Mosquitto)
* Zigbee2MQTT
* ESP32 (WiFi)
* Dispositivos Zigbee (sensores, enchufes)
* Tiras LED direccionables

## Arquitectura

Servidor Ubuntu con Home Assistant en Docker y broker MQTT.
Dispositivos Zigbee integrados mediante Zigbee2MQTT.
ESP32 conectados por WiFi.

## Funcionalidades

* Automatización de iluminación
* Control de climatización
* Gestión de enchufes inteligentes
* Integración de sensores de movimiento, luz y temperatura

## Aprendizajes

* Configuración de sistemas Linux
* Integración de dispositivos IoT
* Comunicación mediante MQTT
* Resolución de incidencias en sistemas reales
